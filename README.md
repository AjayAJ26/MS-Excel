# 🍫 Chocolate Sales & Performance Analysis  

*A data analysis project focused on understanding chocolate sales trends, customer preferences, and store performance using Excel (Power Query, Power Pivot, and DAX).*

---

## ❗ Problem Statement  
A chocolate retail business needs better visibility into:  
- Sales performance across different regions and store types  
- Product category demand (Orange Choco,Milk Bars,Fruit & Nut Bars,Almond Choco...ect)  
- Sales person sales by country 
- shpping box affection factors  

Without structured insights, it becomes difficult to optimize inventory, pricing, and expansion strategies.

---

## 📝 Project Overview  
This project analyzes chocolate sales data to uncover meaningful insights that improve business performance and decision-making.

The analysis focuses on:  
1. **Sales Performance Analysis** – Understanding Sales Person Performance according to the Shiping boxes.
2. **Boxes Shhipped Insights** – Evaluating country wise shipped box and month wise.

### 🌟 Impact

- Highlighted sales person performance accors the time 
- Provided country based contribution for sales.
