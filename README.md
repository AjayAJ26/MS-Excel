# 📞 Call Center Performance Analysis  

*A data analysis project focused on evaluating call center operations, agent performance, and customer satisfaction using Excel (Power Query, Power Pivot, and DAX).*

---

## ❗ Problem Statement  
A call center organization lacks clear visibility into:  
- Agent performance and productivity  
- Call handling efficiency and resolution time  
- Customer satisfaction levels  
- Key operational metrics impacting service quality  

Without structured insights, it becomes difficult to improve customer experience and optimize workforce performance.

---

## 📝 Project Overview  
This project analyzes call center data to uncover actionable insights that enhance operational efficiency and service quality.

The analysis focuses on:  
1. **Call Performance Analysis** – Understanding call volume, handling time, and resolution trends  
2. **Agent & Customer Insights** – Evaluating agent efficiency and customer satisfaction

## 📊 Call Center Analysis  

### ✅ Solution  
- Built an interactive **Call Center Dashboard** in Excel  
- Cleaned and transformed data using **Power Query**  
- Developed data models using **Power Pivot**  
- Used **DAX** for KPI calculations  

### 🌟 Impact  
- Identified top-performing and underperforming agents  
- Highlighted peak call hours and workload distribution  
- Improved understanding of customer satisfaction trends  
- Provided insights for workforce planning and training  

### 📌 Key Metrics  
- 📞 **Total Calls Handled**  
- ⏱️ **Average Call Duration**  
- ✅ **weekly triend**  
- 😊 **Customer Satisfaction Score**  
- 👨‍💼 **Agent Performance Metrics**
-    **Sales Amount**
