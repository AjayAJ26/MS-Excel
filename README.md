# 📊 Excel Basics: Data Cleaning & Core Functions  

*A quick overview of essential Excel features used for data cleaning, transformation, and basic analysis.*

---

## 📝 Overview  
Microsoft Excel is a powerful tool for handling and analyzing data.  
This guide highlights commonly used features and functions that help in preparing and working with data efficiently.

---

## 🧹 Data Cleaning  

### 🔧 Common Tasks  
- Removing duplicates  
- Handling missing values  
- Fixing inconsistent formatting  
- Removing extra spaces  
- Standardizing text  
- Splitting and merging columns  
- Correcting data types  

### 📌 Functions Used  
- `TRIM()`  
- `CLEAN()`  
- `PROPER()`  
- `UPPER()`  
- `LOWER()`  

---

## 🔍 Lookup Functions  

- `VLOOKUP()`  
- `XLOOKUP()`  
- `INDEX()`  
- `MATCH()`  

---

## 🔤 Text Functions  

- `LEFT()`  
- `RIGHT()`  
- `MID()`  
- `LEN()`  
- `CONCAT()`  
- `TEXTJOIN()`  

---

## ⚙️ Logical Functions  

- `IF()`  
- `IFERROR()`  

---

## 📊 Aggregation Functions  

- `SUM()`  
- `AVERAGE()`  
- `COUNT()`  
- `COUNTA()`  

---

## 📈 Applications  

- Data cleaning and preparation  
- Data transformation  
- Basic data analysis  
- Reporting and dashboards

## 🎯 Key Takeaways
