# 🛒 Blinkit Grocery Sales & Performance Analysis  

*A data analysis project focused on understanding grocery sales, customer behavior, and outlet performance using Excel (Power Query, Power Pivot, and DAX).*

---

## ❗ Problem Statement  
Blinkit (online grocery platform) needs better visibility into:  
- Sales performance across different outlet types and locations  
- Product category demand and contribution  
- Customer purchasing patterns  
- Key factors affecting revenue and growth  

The lack of structured insights makes it difficult to optimize operations and improve decision-making.

---

## 📝 Project Overview  
This project analyzes Blinkit grocery data to uncover meaningful insights that help improve business performance.

The analysis focuses on:  
1. **Sales Performance Analysis** – Understanding revenue trends and product demand  
2. **Outlet & Customer Insights** – Evaluating outlet types, sizes, and customer preferences  

---

## 🔢 Data Sources  
The dataset includes:  
- 🏬 Outlet Information (Type, Size, Location)  
- 📦 Product Details (Category, Fat Content, Item Type)  
- 💰 Sales Data (Revenue, Ratings, Sales Volume)  

---

## 📊 Sales & Performance Analysis  

### ✅ Solution  
- Built an interactive **Blinkit Sales Dashboard** in Excel  
- Cleaned and transformed data using **Power Query**  
- Created data models using **Power Pivot**  
- Used **DAX** for calculated metrics  

### 🌟 Impact  
- Identified high-performing products and categories  
- Highlighted sales trends across different outlet types  
- Provided insights for improving product placement and sales strategy  

### 📌 Key Metrics  
- 📈 **Total Sales**  
- 🛍️ **Average Sales per Item**  
- ⭐ **Average Customer Rating**  
- 📦 **Number of Items Sold**  
- 🏬 **Sales by Outlet Type & Location**  

---

## ❓ Business Questions Answered  
- Which product categories generate the highest revenue?  
- Which outlet types perform best in terms of sales?  
- How does location impact sales performance?  
- What is the relationship between item ratings and sales?  
- Which products have high demand but low ratings?  

---

## 🏪 Outlet Analysis  

### ✅ Insights  
- Comparison of **Tier 1, Tier 2, and Tier 3 cities**  
- Performance across different **outlet sizes (Small, Medium, Large)**  
- Sales contribution by **outlet types**  

### 🌟 Value  
- Helps businesses decide **where to expand**  
- Identifies **high-performing outlet formats**  
- Supports **location-based strategy decisions**
